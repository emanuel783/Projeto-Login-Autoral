# Tela de Login - Front-End 

Projeto de uma tela de login desenvolvida juntamente com o professor [Leonardo Rocha](https://github.com/leornadossrocha) utilizando HTML e CSS, com o foco no aprendizado dos fundamentos de desenvolvimento Front-End.

# Descrição 

Este projeto foi desenvolvido com o objetivo de praticar os conceitos básicos de desenvolvimento Front-End.
A aplicação consiste em uma tela de login construída do zero, utilizando apenas HTML E CSS sem o uso de frameworks.

# Tecnologias Utilizadas 
 - HTML5
 - CSS
 - Visual Studio Code

 # Funcionalidades 

- Estrutura de formulário de login
- Campos para E-mail e senha 
- Estilização com CSS 
- Layout organizado e responsivo
